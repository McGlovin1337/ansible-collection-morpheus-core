Examples

.. code-block:: yaml+jinja

      morpheus.core.ssl_certificate_info:

    - name: Get Specific Certificate by Id
      morpheus.core.ssl_certificate_info:

    - name: Get Certificates matching Regular Expression
      morpheus.core.ssl_certificate_info:
        name: ^Web.*$
        regex_name: true
